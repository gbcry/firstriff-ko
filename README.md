# 걸즈 밴드 크라이 First Riff - 비공식 한글화 팬 사이트

> **걸즈 밴드 크라이** 시리즈의 게임, **걸즈 밴드 크라이 First Riff**의 공식 웹사이트인 [[Girls Band Cry First Riff]](gbc-firstriff.com)의 콘텐츠를 비공식 한국어 번역하여 제공하는 웹 서비스 프로젝트입니다.
> 공식 사이트의 정보뿐만 아니라, 국내 유저들이 함께 즐기고 알아두면 좋은 **걸즈 밴드 크라이** 시리즈 전반의 요소를 추가하여 통합적인 정보를 제공합니다.
